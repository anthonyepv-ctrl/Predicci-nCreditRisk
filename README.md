# Sistema de Clasificación de Riesgo Crediticio
## Descripción
Modelo Random Forest entrenado con el dataset German Credit (UCI).
Clasifica clientes como morosos o no morosos.

## Tecnologías
- Python, scikit-learn, imbalanced-learn
- SMOTE para balanceo de clases
- Streamlit para interfaz web

## Uso
Sube un archivo CSV con los datos del cliente en la app web.
