# Codigos German Credit

Guia de codigos categoricos usados por el dataset German Credit y por la entrada manual de la app.

## Estado de cuenta

| Codigo | Significado |
| --- | --- |
| A11 | Cuenta corriente < 0 DM |
| A12 | Cuenta corriente entre 0 y 200 DM |
| A13 | Cuenta corriente >= 200 DM |
| A14 | Sin cuenta corriente |
